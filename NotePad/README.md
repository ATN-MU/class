# NotePad
## **基本功能**

1. 笔记列表显示笔记条目的修改时间
2. 笔记内容的搜索功能
3. 笔记卡片的添加、删除、修改，更新最后修改时间。
4. 每个便签卡片包含的信息：标题、内容、时间日期。
